## 学习思路
#### 前置知识
[git使用方法](https://github.com/996-PYG/QYN./tree/master/Git)  
#### 每日流程  
1. 在自己的分支上提交学习笔记等，commit并及时推送到远程的对应分支上。  
2. 切换到对方的分支，更新，有问题则进行修改并推送到远程分支，或者打开GitHub留言。  
3. 更新本地所有分支，确认无误后将自己的分支与主分支进行一次合并，并将主分支推送到GitHub
