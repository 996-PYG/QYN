Release Notes for STM32G0C1E-EV Board drivers

Copyright © 2020 STMicroelectronics

Purpose

This driver provides a set of functions to manage:

  • LEDs, Buttons, SD card, temperature sensor, LCD, MUX and PWR on STM32G0C1E-EV board from STMicroelectronics

Update History

Main Changes

  • Update BSP to support USB-PD FRS feature.

Supported Devices and Boards

  • STM32G0C1xx/B1xx devices
  • STM32G0C1E-EV mother board (MB1581B)
  • “Legacy” daughter board (MB1351A)
  • USBPD daughter board (MB1352C)

Main Changes

  • Increase VBUS voltage for 5V config to support the load at 3A.
  • Remove extra timing to fix an detecting issue on Ellisys TC 4.2.3
  • Update Licensing information and format
    • Add LICENSE.txt file
    • Remove License message from Release_Notes.html
    • Update header files with new license format

Supported Devices and Boards

  • STM32G0C1xx/B1xx devices
  • STM32G0C1E-EV mother board (MB1581B)
  • “Legacy” daughter board (MB1351A)
  • USBPD daughter board (MB1352C)
